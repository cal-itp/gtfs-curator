# gtfs-curator
Curated collections of GTFS data products.

This repository is home to Caltran's Division of Data & Digital Services
stable, mature and maintained GTFS data products.
Many started out in the [data-analyses](https://github.com/cal-itp/data-analyses) 
repo. New data products will launch here as we aim to make the GTFS 
schedule and real-time performance metrics publicly available.
